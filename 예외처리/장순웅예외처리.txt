예외처리  목록 : http 500
화면 : 판매계획등록
증상 : 필수 입력 요소에 null값을 넣은 채로 저장버튼 클릭시 500에러
해결방안 : if - else 문을 사용하여 필수입력요소에 빈 값이 있을경우 메세지 출력 후 해당 빈칸으로 커서이동
